# Compact Go Neural Network (Under 100k Parameters)

This project presents the design, training, and evaluation of compact
neural network architectures for the game of Go (19×19), developed under
a strict constraint of fewer than **100,000 trainable parameters**.

The objective is to build an efficient policy--value network capable of
predicting both: - the next move (policy head) - the probability
of winning (value head)

The final architecture combines ConvNeXt-inspired blocks and lightweight
attention mechanisms, achieving strong performance while remaining
compact.

## Architectures Overview

Five successive architectures were explored:

1.  Compact Residual Policy--Value Network\
2.  Simplified AlphaZero-style Architecture\
3.  Hybrid Convolution + Local Attention\
4.  ConvNeXt + Attention-Enhanced Architecture\
5.  Optimized ConvNeXt Hybrid (Final Model) 

The final model: - Uses 6 Hybrid ConvNeXt blocks\
- Integrates Local Channel Attention, Squeeze-and-Excitation, and Global
Context modules\
- Contains **98,167 trainable parameters**\
- Was trained up to **10,000 epochs**

------------------------------------------------------------------------

## Training Details

-   Optimizer: Adam (learning rate = 1e-3)
-   Loss:
    -   Policy: Categorical Cross-Entropy
    -   Value: Mean Squared Error (weighted by 0.4)
-   Batch size: 128
-   Data augmentation: Random board symmetries (dihedral group D₄)
-   Dataset: Self-play games generated by KataGo
-   Data interface: `golois` Python module

## Results

The final architecture achieves:

-   **Policy Accuracy:** \~50%
-   **Value MAE:** \~0.20
-   Stable convergence across 10,000 epochs
